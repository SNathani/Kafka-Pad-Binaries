Getting Started

  • Install Kafka

    Follow the instructions in this article to intall Kafka and ZooKeeper from One-to-run-them-all docker compose file.

  • Connecting to Kafka

    1. Click on the 'Add New' button to create a new connection row
    2. Enter a connection name in the Name column and the Kafka broker name and port information in the Brokers column. Multiple brokers can be added by separating them with a comma. Press TAB to commit changes.
    3. Click on the 'Connect' button to connect to the brokers selected

  • Add/Remove Topics (Tab: TOPICS)

    1. To add a topic, enter the topic name in the search box.
    2. If the topic name is not found, the Plus button will be enabled. Click on the Plus button to add the topic with the name in the search box.
    3. To remove a topic, select one or more topic checkboxes and hit the 'Trash Can' button.

    Note: Topic names are case-sensitive

  • Consuming Messages from Topics (Tab: CONSUMERS)

    1. Add a consumer by clicking the Plus button.
      • Select the topics to consume and click the 'Subscribe to Topics' button
      • Enter the Client Id and Group Id information
      • Click on the OK button to add a consumer

  • Sending Message to Topics (Tab: PRODUCERS)

    1. Add a producer by clicking on the Plus button.
      • Select one or more topics from the list and click on the 'Add Selected Topics' button to add the selected topics to the subcription list
      • Enter Client Id
      • Click on the OK button to finish adding the producer
    2. To send messages to a topic
      • Select the producer that is associated with the topic
      • Enter the message in the text box and click the 'Send' button from the above toolbar
